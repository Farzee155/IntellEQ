Semester Project
IntellEQ

Intelligent equalization for audio mastering using neural networks trained by a dataset of audio files assorted by genre.

Repository contains:
- standalone app install file
- Training dataset of audio files
- Dataset spreadsheet for use with neural network
- PyCharm project to create neural net using Tennis
- MATLAB data processing scripts for filtering and analyzing audio
- Proposal Abstract
- Paper
