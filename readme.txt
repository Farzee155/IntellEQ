Semester Project
IntellEQ

Intelligent equalization for audio mastering using neural networks trained by a dataset of audio files assorted by genre.

Repository contains:
- standalone app install file (IntellEQApp.mlappinstall)
- Training dataset of audio files (Genre Dataset)
- Dataset spreadsheet for use with neural network (IntellEQ Training and Testing Data.xlsx)
- PyCharm project to create neural net using Tennis (IntellEQ-neural-net-training)
- MATLAB data processing scripts for filtering and analyzing audio (MATLAB Data Processing)
- Proposal Abstract
- Final Paper
